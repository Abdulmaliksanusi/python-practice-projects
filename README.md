# Portfolio Website

## Overview
A personal portfolio website built to present technical skills, project work, and contact information in a clear web format. The website demonstrates front-end development skills through multiple pages, styling, layout, and basic interactivity.

## Tools Used
- HTML
- CSS
- JavaScript

## Features
- Homepage introducing the portfolio
- About page with background information
- Services or skills page
- Contact page
- Responsive page structure
- Organised CSS styling
- Basic JavaScript functionality

## Skills Demonstrated
- Web page structure using HTML
- Styling and layout using CSS
- Basic JavaScript
- Multi-page website development
- Navigation design
- User-focused presentation

## Project Outcome
Delivered a functional portfolio website that can be used to showcase software development projects, web development skills, and placement application evidence.
